## 흐름
1. 배열의 인덱스를 활용한 카운터 도입하면 쉽게 해결

---

## 코드
```java
import java.util.Scanner;
import java.io.FileInputStream;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);
        int T;
        T=sc.nextInt();

        for(int test_case = 1; test_case <= T; test_case++)
        {
            int t = sc.nextInt();
            
            int[] arr = new int[101];

            for(int i = 0; i < 1000; i++){
                int input = sc.nextInt();
                arr[input]++;
            }
            
            int max = 0;
            int index = 0;

            for(int i = 1; i < arr.length; i++){
                if(arr[i] >= max){
                    max = arr[i];
                    index = i;
                }
            }

            System.out.println("#" + t + " " + index);
        }
    }
}
```

## AI 코드 리뷰

### 좋았던 점

1. 카운팅 배열 사용
    - 점수 범위가 `0 ~ 100`이라 `int[] arr = new int[101]`로 푼 방식이 적절하다

2. 입력과 카운팅을 동시에 처리
    - 점수를 따로 저장하지 않고 바로 `arr[input]++` 한 점이 좋다

3. 동점 처리
    - `arr[i] >= max`를 사용해서 같은 빈도일 때 더 큰 점수로 갱신한 점이 좋다

---

### 개선할 점

1. 변수명 개선
    - `arr`보다는 `count`가 의미가 더 명확하다
    - `index`보다는 `answer` 또는 `modeScore`가 더 자연스럽다

2. 출력 번호 주의
    - `test_case`가 아니라 입력받은 `t`를 출력해야 한다

3. 반복 시작점
    - 점수는 `0점`도 가능하므로 원칙적으로 `i = 0`부터 확인하는 게 더 정확하다
    - 다만 현재는 `max = 0`, `index = 0`이라 큰 문제는 없다

---

## AI 개선 코드

```java
import java.util.Scanner;

class Solution {
    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();

        for (int test_case = 1; test_case <= T; test_case++) {
            int caseNumber = sc.nextInt();

            int[] count = new int[101];

            for (int i = 0; i < 1000; i++) {
                int score = sc.nextInt();
                count[score]++;
            }

            int maxCount = 0;
            int modeScore = 0;

            for (int score = 0; score <= 100; score++) {
                if (count[score] >= maxCount) {
                    maxCount = count[score];
                    modeScore = score;
                }
            }

            System.out.println("#" + caseNumber + " " + modeScore);
        }
    }
}
```
