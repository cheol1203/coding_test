# SWEA 1926 👏 간단한 369게임 (Java)

## 느낀점
>
- 처음에는 자리수를 중첩 if로 처리해서 일부 케이스를 놓쳤다
- 특히 `30`, `60`, `90` 같은 숫자를 놓친 경험이 핵심 포인트였다
- 이후 `count`로 박수 개수를 세는 구조로 바꾸면서 로직이 훨씬 명확해졌다
- 숫자 분해 방식도 가능하지만 문자열 방식이 더 직관적이라는 걸 느낌

---

## 흐름
1. `1`부터 `N`까지 반복
2. 각 자리수에서 `3`, `6`, `9` 개수 세기
3. 개수 > 0 → `-` 출력
4. 아니면 숫자 출력
5. 공백 처리

---

## 코드
```java
import java.util.Scanner;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();

        for(int test_case = 1; test_case <= T; test_case++)
        {
            int third_num = test_case / 100;
            int second_num = (test_case % 100) / 10;
            int first_num = (test_case % 100) % 10;

            int count = 0;

            if(third_num % 3 == 0 && third_num != 0){
                count++;
            }
            if(second_num % 3 == 0 && second_num != 0){
                count++;
            }
            if(first_num % 3 == 0 && first_num != 0){
                count++;
            }

            if(count > 0){
                for(int i = 0; i < count; i++){
                    System.out.print("-");
                }
            } else {
                System.out.print(test_case);
            }

            if(test_case != T){
                System.out.print(" ");
            }
        }
    }
}
```

## 질문 & 실수 정리

### 새로 배운 것

- 조건을 **중첩이 아니라 독립적으로 검사해야 한다**
- 박수 개수를 `count`로 누적하는 방식이 더 안전하고 확장성이 좋다

---

### 실수했던 부분

- 자리수를 중첩 if로 처리해서 일부 케이스 누락 (`30`, `60`, `90`)
- `&`와 `&&` 차이를 정확히 구분하지 못함
- 자리수 분해 방식에만 집착해서 코드가 길어짐

---

## AI 코드 리뷰

### 개선할 점

1. 자리수 처리 방식 개선
   - 숫자 분해 방식 → 문자열 방식이 더 직관적

2. 확장성 부족
   - 현재는 3자리까지만 자연스럽게 처리됨

3. 가독성 개선
   - 문자열로 바꾸면 코드가 더 짧고 명확해짐

---

## AI 개선 코드

```java
import java.util.Scanner;

class Solution {
    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);

        int N = sc.nextInt();

        for (int num = 1; num <= N; num++) {
            String str = String.valueOf(num);
            int count = 0;

            for (int i = 0; i < str.length(); i++) {
                char c = str.charAt(i);

                if (c == '3' || c == '6' || c == '9') {
                    count++;
                }
            }

            if (count > 0) {
                for (int i = 0; i < count; i++) {
                    System.out.print("-");
                }
            } else {
                System.out.print(num);
            }

            if (num != N) {
                System.out.print(" ");
            }
        }
    }
}
```
