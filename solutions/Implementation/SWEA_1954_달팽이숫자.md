# SWEA 1954 🐌 달팽이 숫자 (Java)

## 느낀점
>
- 매우 어려웠던 D2 문제
- 손도 못대서 AI 풀이를 참고해서 풀었다.

---

## 흐름
1. `N x N` 크기의 배열을 만든다.
2. 시작 위치를 `(0, 0)`으로 잡고, 처음 방향은 오른쪽으로 설정한다.
3. 1부터 `N * N`까지 반복하면서 현재 위치에 숫자를 넣는다.
4. 다음 칸이 배열 범위를 벗어나거나 이미 숫자가 있으면 방향을 바꾼다.
5. 바뀐 방향으로 이동하며 배열을 채운다.
6. 테스트 케이스 번호와 함께 배열을 출력한다.

---

## 코드
```java
import java.util.Scanner;

class Solution {
    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);

        int T = sc.nextInt();

        for (int test_case = 1; test_case <= T; test_case++) {
            int n = sc.nextInt();
            int[][] arr = new int[n][n];

            // 방향 설정 배열
            int[] dr = {0, 1, 0, -1};
            int[] dc = {1, 0, -1, 0};

            int r = 0;
            int c = 0;
            int dir = 0; //조건 만족시 방향을 바꾸는 변수

            for (int i = 1; i <= n * n; i++) {
                arr[r][c] = i;

                int nr = r + dr[dir];
                int nc = c + dc[dir];

                if (nr < 0 || nr >= n || nc < 0 || nc >= n || arr[nr][nc] != 0) {
                    dir = (dir + 1) % 4;
                }

                r = r + dr[dir];
                c = c + dc[dir];
            }

            System.out.println("#" + test_case);

            for (int i = 0; i < n; i++) {
                for (int j = 0; j < n; j++) {
                    System.out.print(arr[i][j] + " ");
                }
                System.out.println();
            }
        }
    }
}
```
