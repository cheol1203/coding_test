# SWEA 2805 🌾 농작물 수확하기 (Java)

## 느낀점
>
- 처음에는 위쪽 마름모, 아래쪽 마름모 나누어서 더하려고 했으나 그러면 아래쪽 식이 너무 복잡해짐
- 절대값을 활용하면 한번에 계산할 수 있다!

---

## 흐름
1. 테스트케이스 개수 `T` 입력
2. 농장 크기 `N` 입력
3. `N x N` 농장 정보를 문자열로 한 줄씩 입력
4. 각 문자를 숫자로 변환해서 배열에 저장
5. 가운데 인덱스 `mid = N / 2` 계산
6. 각 행마다 가운데 행과의 거리 `begin = Math.abs(mid - i)` 계산
7. 수확 범위는 `begin`부터 `N - 1 - begin`까지
8. 해당 범위의 값을 모두 더한 뒤 출력

---
## 1차 오답 코드
```java
import java.util.Scanner;
import java.io.FileInputStream;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);
        int T;
        T=sc.nextInt();

        for(int test_case = 1; test_case <= T; test_case++)
        {
            int n = sc.nextInt();

            int[][] arr = new int[n][n];

            for(int i =0; i<n; i++) {
                for(int j = 0; j<n; j++) {
                    arr[i][j]= sc.nextInt();
                }
            }

            int result = 0;
            for(int i = 0; i<= n/2; i++) {
                for(int j = 0; j < 2*i+1; j++) {
                    result += arr[i][n/2-i+j];
                }
            }

            for(int i = n/2+1; i<n; i++) {
                for(int j = 0; j < (-2)*i + (2*n - 1); j++) {
                    result += arr[i][i-((1/2)*n-(1/2))+j];
                }
            }

            System.out.println("#" + test_case + " " + result);
        }
    }
}
```
## 제출 코드

```java
import java.util.Scanner;
import java.io.FileInputStream;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);
        int T;
        T=sc.nextInt();

        for(int test_case = 1; test_case <= T; test_case++)
        {
            int n = sc.nextInt();

            int[][] arr = new int[n][n];

            for(int i = 0; i < n; i++) {
                String str = sc.next();

                for(int j = 0; j < n; j++) {
                    arr[i][j] = str.charAt(j) - '0';
                }
            }

            int result = 0;
            int mid = n / 2;

            for(int i = 0; i < n; i++) {
                int begin = Math.abs(mid - i);
                int end = n - 1 - begin;

                for(int j = begin; j <= end; j++) {
                    result += arr[i][j];
                }
            }

            System.out.println("#" + test_case + " " + result);
        }
    }
}

```
## 질문 & 실수 정리

### 새로 배운 것

- 공백 없이 붙어 있는 숫자는 `nextInt()`가 아니라 `String`으로 입력받아야 한다
- 문자 숫자를 실제 정수로 바꾸려면 `'0'`을 빼면 된다
  - 예: `'7' - '0' = 7`
- 마름모 영역은 가운데 행과의 거리로 구할 수 있다
- `Math.abs(mid - i)`를 사용하면 위쪽/아래쪽을 나누지 않고 한 번에 처리할 수 있다

---

### 실수했던 부분

- 처음에는 농장 값을 `nextInt()`로 입력받으려고 했다
  - 하지만 입력이 `14054`처럼 공백 없이 들어오기 때문에 한 자리씩 읽을 수 없다
- 위쪽 영역과 아래쪽 영역을 따로 계산하려고 해서 식이 복잡해졌다
- 마름모 범위는 복잡한 수식보다 `Math.abs()`를 사용하는 방식이 더 안전하다

### 개선할 점

1. 배열 없이도 풀이 가능
   - 현재는 `arr`에 저장한 뒤 다시 순회하지만, 각 줄을 입력받을 때 바로 합산할 수도 있다

2. 변수명 개선
   - `arr` → `farm`
   - `begin` → `start`
   - `result` → `sum` 또는 `answer`

## AI 개선 코드

```java
import java.util.Scanner;

class Solution {
    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();

        for (int test_case = 1; test_case <= T; test_case++) {
            int n = sc.nextInt();
            int mid = n / 2;
            int answer = 0;

            for (int row = 0; row < n; row++) {
                String line = sc.next();

                int start = Math.abs(mid - row);
                int end = n - 1 - start;

                for (int col = start; col <= end; col++) {
                    answer += line.charAt(col) - '0';
                }
            }

            System.out.println("#" + test_case + " " + answer);
        }
    }
}
```
