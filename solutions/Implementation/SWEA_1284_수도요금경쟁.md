# SWEA 1284 🚰 수도 요금 경쟁 (Java)

## 느낀점
>
- 이거 못 풀면 전공자 아니다...

---

## 흐름
1. 테스트케이스 개수 `T` 입력
2. 각 테스트케이스마다 `P, Q, R, S, W` 입력
3. A사 요금 계산: `P * W`
4. B사 요금 계산
    - `W <= R`이면 `Q`
    - `W > R`이면 `Q + (W - R) * S`
5. 두 요금 중 더 작은 값 출력

---

## 코드
```java
import java.util.Scanner;
import java.io.FileInputStream;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);
        int T;
        T=sc.nextInt();

        for(int test_case = 1; test_case <= T; test_case++)
        {
            int p = sc.nextInt();
            int q = sc.nextInt();
            int r = sc.nextInt();
            int s = sc.nextInt();
            int w = sc.nextInt();

            int a_price = p * w;
            int b_price = 0;

            if(w <= r){
                b_price = q;
            }else{
                b_price = q + s * (w - r);
            }

            int price = Math.min(a_price, b_price);

            System.out.println("#" + test_case + " " + price);
        }
    }
}
```

## AI 코드 리뷰

### 좋았던 점

1. 문제 조건을 정확히 나눔
   - `w <= r`일 때와 초과할 때를 잘 구분했다

2. 요금 계산을 분리함
   - A사 요금과 B사 요금을 각각 계산한 뒤 비교해서 흐름이 명확하다

3. `Math.min()` 사용
   - 두 요금 중 작은 값을 구하는 방식이 간단하고 적절하다

---

### 개선할 점

1. 변수명 개선
   - 알고리즘 문제에서는 `p, q, r, s, w`도 괜찮지만, 복습용 기록에서는 의미 있는 변수명이 더 좋다

2. `b_price` 초기값 제거 가능
   - 조건문에서 반드시 값이 할당되므로 초기값 `0`은 없어도 된다

3. 카멜 케이스 사용
   - Java에서는 보통 `a_price`보다 `aPrice`처럼 작성한다

## AI 개선 코드

```java
import java.util.Scanner;

class Solution {
    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();

        for (int test_case = 1; test_case <= T; test_case++) {
            int pricePerLiterA = sc.nextInt();
            int basePriceB = sc.nextInt();
            int baseLimitB = sc.nextInt();
            int extraPriceB = sc.nextInt();
            int usage = sc.nextInt();

            int aPrice = pricePerLiterA * usage;
            int bPrice;

            if (usage <= baseLimitB) {
                bPrice = basePriceB;
            } else {
                bPrice = basePriceB + (usage - baseLimitB) * extraPriceB;
            }

            int answer = Math.min(aPrice, bPrice);

            System.out.println("#" + test_case + " " + answer);
        }
    }
}
```
