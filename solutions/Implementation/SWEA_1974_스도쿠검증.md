# SWEA 1974 🧩 스도쿠 검증 (Java)

## 느낀점
>
- 행과 열 중복검사는 구현했지만 3x3 박스 중복검사를 구현하지 못해 ai의 도움을 받았다.
- 기존 행과 열 중복검사 때는 3중for문을 사용했지만 `bool` 배열을 통해 2중 for문으로 구현할 수 있었다.
- 중복 검사에는 `boolean[]` 배열을 사용하는 방식이 훨씬 깔끔했다
- `boolean[10]`을 만들어서 숫자 `1~9`를 인덱스로 바로 사용하는 방식이 직관적이었다
- 3x3 박스는 시작 좌표를 `0, 3, 6`으로 이동시키는 게 핵심이었다

---

## 흐름
1. 테스트케이스 개수 `T` 입력
2. `9 x 9` 스도쿠 배열 입력
3. 각 행에 중복 숫자가 있는지 검사
4. 각 열에 중복 숫자가 있는지 검사
5. 각 `3 x 3` 박스에 중복 숫자가 있는지 검사
6. 중복이 하나라도 있으면 `0`
7. 모두 통과하면 `1` 출력

---

## 1차 제출코드
```java
import java.util.Scanner;
import java.io.FileInputStream;

class Solution
{
	public static void main(String args[]) throws Exception
	{
		Scanner sc = new Scanner(System.in);
		int T;
		T=sc.nextInt();

		for(int test_case = 1; test_case <= T; test_case++)
		{
            int[][] arr = new int[9][9];
            
            for(int i = 0; i< 9; i++){
            	for(int j = 0; j< 9; j++){
                	arr[i][j]=sc.nextInt();
                }
            }
            
            for(int i = 0; i<9; i++){
            	for(int j = 0; j< 9; j++){
            		for(int k = j+1; k< 9; k++){
                		if(arr[i][j]==arr[i][k]){
                    		System.out.println(0);
                        	break;
                    	}
                	}
            	}
            }
            
            for(int i = 0; i<9; i++){
            	for(int j = 0; j< 9; j++){
            		for(int k = j+1; k< 9; k++){
                		if(arr[j][i]==arr[k][i]){
                    		System.out.println(0);
                        	break;
                    	}
                	}
            	}
            }
		}
	}
}
```

## 최종 제출코드
```java
import java.util.Scanner;
import java.io.FileInputStream;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);
        int T;
        T=sc.nextInt();

        for(int test_case = 1; test_case <= T; test_case++)
        {
            int[][] arr = new int[9][9];
            int result = 1;

            for(int i = 0; i < 9; i++){
                for(int j = 0; j < 9; j++){
                    arr[i][j] = sc.nextInt();
                }
            }

            for(int i = 0; i < 9; i++){
                boolean[] boo = new boolean[10];

                for(int j = 0; j < 9; j++){
                    int num = arr[i][j];

                    if(boo[num]){
                        result = 0;
                    }

                    boo[num] = true;
                }
            }

            for(int i = 0; i < 9; i++){
                boolean[] boo = new boolean[10];

                for(int j = 0; j < 9; j++){
                    int num = arr[j][i];

                    if(boo[num]){
                        result = 0;
                    }

                    boo[num] = true;
                }
            }

            for(int i = 0; i < 9; i += 3){
                for(int j = 0; j < 9; j += 3){
                    boolean[] boo = new boolean[10];

                    for(int k = i; k < i + 3; k++){
                        for(int w = j; w < j + 3; w++){
                            int num = arr[k][w];

                            if(boo[num]){
                                result = 0;
                            }

                            boo[num] = true;
                        }
                    }
                }
            }

            System.out.println("#" + test_case + " " + result);
        }
    }
}
```

## 질문 & 실수 정리

### 새로 배운 것

- 숫자 중복 여부는 `boolean[]` 배열로 쉽게 확인할 수 있다
- `boolean[10]`을 사용하면 숫자 `1~9`를 그대로 인덱스로 사용할 수 있다
- 3x3 박스의 시작 좌표는 `0, 3, 6` 단위로 이동한다

---

### 실수했던 부분

- 중복을 발견했을 때 바로 `System.out.println(0)`을 출력하려고 했다
  - break로는 for문을 하나밖에 탈출 못한다.
  - 그래서 `result = 0`으로 상태만 저장하는 방식이 좋다
- 3x3 박스 검사에서 안쪽 반복문의 증감식을 잘못 작성했었다
  - `j++`가 아니라 `w++`가 되어야 했다

---

## AI 코드 리뷰

### 좋았던 점

1. `boolean[]`을 사용한 중복 검사
   - 숫자가 이미 등장했는지 빠르게 확인할 수 있어서 적절한 방식이다

2. 행 / 열 / 3x3 박스를 분리해서 검사
   - 검사 대상이 명확하게 나뉘어 있어서 흐름을 이해하기 쉽다

3. 출력을 마지막에 한 번만 처리
   - `result` 변수를 사용해서 정답 상태를 저장한 점이 좋다

4. 3x3 박스 시작점 처리
   - `i += 3`, `j += 3`으로 박스 시작 위치를 잘 잡았다

---

### 개선할 점

1. 변수명 개선
   - `arr` → `sudoku`
   - `boo` → `visited`
   - `result` → `answer`

2. 중복 발견 시 불필요한 검사 줄이기
   - 현재는 이미 `result = 0`이어도 끝까지 검사한다
   - 문제 크기가 작아서 괜찮지만, 함수로 분리하면 더 깔끔하게 조기 종료할 수 있다

3. 검사 로직 함수화
   - 행, 열, 박스 검사를 함수로 나누면 가독성이 더 좋아진다

## AI 개선 코드

```java
import java.util.Scanner;

class Solution {
    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();

        for (int test_case = 1; test_case <= T; test_case++) {
            int[][] sudoku = new int[9][9];

            for (int row = 0; row < 9; row++) {
                for (int col = 0; col < 9; col++) {
                    sudoku[row][col] = sc.nextInt();
                }
            }

            int answer = isValidSudoku(sudoku) ? 1 : 0;

            System.out.println("#" + test_case + " " + answer);
        }
    }

    static boolean isValidSudoku(int[][] sudoku) {
        for (int i = 0; i < 9; i++) {
            if (!checkRow(sudoku, i)) {
                return false;
            }

            if (!checkCol(sudoku, i)) {
                return false;
            }
        }

        for (int row = 0; row < 9; row += 3) {
            for (int col = 0; col < 9; col += 3) {
                if (!checkBox(sudoku, row, col)) {
                    return false;
                }
            }
        }

        return true;
    }

    static boolean checkRow(int[][] sudoku, int row) {
        boolean[] visited = new boolean[10];

        for (int col = 0; col < 9; col++) {
            int num = sudoku[row][col];

            if (visited[num]) {
                return false;
            }

            visited[num] = true;
        }

        return true;
    }

    static boolean checkCol(int[][] sudoku, int col) {
        boolean[] visited = new boolean[10];

        for (int row = 0; row < 9; row++) {
            int num = sudoku[row][col];

            if (visited[num]) {
                return false;
            }

            visited[num] = true;
        }

        return true;
    }

    static boolean checkBox(int[][] sudoku, int startRow, int startCol) {
        boolean[] visited = new boolean[10];

        for (int row = startRow; row < startRow + 3; row++) {
            for (int col = startCol; col < startCol + 3; col++) {
                int num = sudoku[row][col];

                if (visited[num]) {
                    return false;
                }

                visited[num] = true;
            }
        }

        return true;
    }
}
```
