# SWEA 1215 🔁 회문1 (Java)

## 느낀점
>
- 포기할뻔..
- 3중 for문으로 길이의 절반만큼만 탐색하여 양쪽 문자가 같은지 비교
- 회문은 양쪽 끝에서부터 가운데 방향으로 비교하면 된다는 점을 다시 확인했다
- 문자열을 한 줄씩 입력받고 `charAt()`으로 문자 배열에 저장하는 방식이 익숙해졌다

---

## 흐름
1. 테스트케이스는 총 10개이므로 10번 반복
2. 회문 길이 `length` 입력
3. `8 x 8` 문자 배열 입력
4. 각 행에서 길이 `length`만큼의 구간을 검사
5. 양끝 문자가 모두 같으면 가로 회문으로 count 증가
6. 각 열에서 길이 `length`만큼의 구간을 검사
7. 양끝 문자가 모두 같으면 세로 회문으로 count 증가
8. 최종 회문 개수 출력

---

## 제출 코드

```java
import java.util.Scanner;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);

        for(int test_case = 1; test_case <= 10; test_case++)
        {
            int length = sc.nextInt();
            char[][] arr = new char[8][8];

            for(int i = 0; i < 8; i++) {
                String str = sc.next();

                for(int j = 0; j < 8; j++) {
                    arr[i][j] = str.charAt(j);
                }
            }

            int count = 0;

            for(int i = 0; i < 8; i++) {
                for(int j = 0; j < 8 - length + 1; j++) {
                    int success = 0;

                    for(int k = 0; k < length / 2; k++) {
                        if(arr[i][j + k] == arr[i][j + length - 1 - k]) {
                            success++;
                        }
                    }

                    if(success == length / 2) {
                        count++;
                    }
                }
            }

            for(int i = 0; i < 8; i++) {
                for(int j = 0; j < 8 - length + 1; j++) {
                    int success = 0;

                    for(int k = 0; k < length / 2; k++) {
                        if(arr[j + k][i] == arr[j + length - 1 - k][i]) {
                            success++;
                        }
                    }

                    if(success == length / 2) {
                        count++;
                    }
                }
            }

            System.out.println("#" + test_case + " " + count);
        }
    }
}
```

## 질문 & 실수 정리


### 실수했던 부분

- `success`를 세는 방식은 동작하지만, 다른 문자가 발견되면 바로 `break`하는 방식이 더 효율적이다

## AI 코드 리뷰

### 개선할 점

1. `success` 대신 boolean 사용
    - 다른 문자를 발견하면 바로 `false`로 바꾸고 `break`하면 더 직관적이다

2. 중복 코드 함수화
    - 가로 검사와 세로 검사는 구조가 비슷하므로 `isRowPalindrome`, `isColPalindrome` 같은 함수로 분리할 수 있다

3. 변수명 개선
    - `arr` → `board`
    - `str` → `line`
    - `success` → `matchCount` 또는 boolean `isPalindrome`

4. 상수 사용
    - `8` 대신 `SIZE = 8`을 사용하면 의미가 더 명확하다

## AI 개선 코드

```java
import java.util.Scanner;

class Solution {
    static final int SIZE = 8;

    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);

        for (int test_case = 1; test_case <= 10; test_case++) {
            int length = sc.nextInt();
            char[][] board = new char[SIZE][SIZE];

            for (int row = 0; row < SIZE; row++) {
                String line = sc.next();

                for (int col = 0; col < SIZE; col++) {
                    board[row][col] = line.charAt(col);
                }
            }

            int count = 0;

            for (int row = 0; row < SIZE; row++) {
                for (int start = 0; start <= SIZE - length; start++) {
                    if (isRowPalindrome(board, row, start, length)) {
                        count++;
                    }
                }
            }

            for (int col = 0; col < SIZE; col++) {
                for (int start = 0; start <= SIZE - length; start++) {
                    if (isColPalindrome(board, start, col, length)) {
                        count++;
                    }
                }
            }

            System.out.println("#" + test_case + " " + count);
        }
    }

    static boolean isRowPalindrome(char[][] board, int row, int start, int length) {
        for (int k = 0; k < length / 2; k++) {
            if (board[row][start + k] != board[row][start + length - 1 - k]) {
                return false;
            }
        }

        return true;
    }

    static boolean isColPalindrome(char[][] board, int start, int col, int length) {
        for (int k = 0; k < length / 2; k++) {
            if (board[start + k][col] != board[start + length - 1 - k][col]) {
                return false;
            }
        }

        return true;
    }
}
```
