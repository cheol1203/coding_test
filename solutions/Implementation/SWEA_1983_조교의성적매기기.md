# SWEA 1983 🧑‍🏫 조교의 성적 매기기 (Java)

## 느낀점
>
- 나중에 다시 풀어볼 문제
- 점수를 계산하여 배열에 넣는 것 까지는 구현했지만 학점 배열과 점수 배열을 매칭 하는 것을 못해 ai코드를 보고 맞췄다.
- `N / 10` 단위로 학점이 나뉜다는 걸 이해하면서 해결했다

---

## 흐름
1. 모든 학생의 총점을 계산한다.
2. K번째 학생의 총점을 따로 저장한다.
3. 총점 기준으로 내림차순 정렬한다.
4. K번째 학생이 몇 등인지 찾는다.
5. 등수에 따라 학점을 출력한다.

---

## 코드
```java
import java.util.Scanner;
import java.io.FileInputStream;
import java.util.Arrays;
import java.util.Collections;

class Solution
{
   public static void main(String args[]) throws Exception
   {
      Scanner sc = new Scanner(System.in);
      int T;
      T=sc.nextInt();
      for(int test_case = 1; test_case <= T; test_case++)
      {
         System.out.print("#" + test_case + " ");
         int n=sc.nextInt();
         int k=sc.nextInt();

         int[][] arr = new int[n][3];
         Double[] score = new Double[n];
         String[] degree = new String[]{"A+","A0","A-","B+","B0","B-","C+","C0","C-","D0"};
         for(int i = 0; i< arr.length; i++){
            for(int j = 0; j< 3; j++){
               arr[i][j]= sc.nextInt();
            }
         }

         for(int i = 0; i< arr.length; i++){
            for(int j = 0; j< 3; j++){
               score[i]= arr[i][0]*((double)35/100) + arr[i][1]*((double)45/100) + arr[i][2]*((double)20/100);
            }
         }


         double target_score = score[k-1];

         Arrays.sort(score, Collections.reverseOrder());

         for(int i =0; i<score.length; i++){
            if(score[i]==target_score){
               int groupSize = n/10;
               System.out.println(degree[i/groupSize]);
               break;
            }
         }
      }
   }
}
```

## 질문 & 실수 정리

### 새로 배운 것

- 역정렬을 할때는 Arrays.sort(arr, Collections.reverseOrder())을 사용한다.

---

### 실수했던 부분

- 점수 계산에서 이중 for문을 사용할 필요가 없다
- 등수를 그대로 쓰는 게 아니라 구간으로 나눠야 한다
- `i / (N / 10)`이 학점 결정의 핵심 로직
- 래퍼 클래스를 사용할 때 equals()비교를 해야한다.

## AI 코드 리뷰

### 개선할 점

1. 불필요한 2차원 배열 제거
   - 점수 입력과 동시에 총점 계산 가능

2. 반복문 단순화
   - 총점 계산은 학생당 한 번만 수행

3. 타입 안정성 개선
   - `double` 대신 정수 계산 사용

4. 비교 방식 개선
   - `==` 대신 정수 비교 또는 `.equals()` 사용

## AI 개선 코드

```java
import java.util.Scanner;
import java.util.Arrays;
import java.util.Collections;

class Solution {
   public static void main(String args[]) throws Exception {
      Scanner sc = new Scanner(System.in);
      int T = sc.nextInt();

      String[] grades = {
              "A+", "A0", "A-",
              "B+", "B0", "B-",
              "C+", "C0", "C-",
              "D0"
      };

      for (int test_case = 1; test_case <= T; test_case++) {
         int n = sc.nextInt();
         int k = sc.nextInt();

         Integer[] scores = new Integer[n];
         int targetScore = 0;

         for (int i = 0; i < n; i++) {
            int midterm = sc.nextInt();
            int finalExam = sc.nextInt();
            int assignment = sc.nextInt();

            int totalScore = midterm * 35 + finalExam * 45 + assignment * 20;
            scores[i] = totalScore;

            if (i == k - 1) {
               targetScore = totalScore;
            }
         }

         Arrays.sort(scores, Collections.reverseOrder());

         int groupSize = n / 10;

         for (int i = 0; i < n; i++) {
            if (scores[i] == targetScore) {
               System.out.println("#" + test_case + " " + grades[i / groupSize]);
               break;
            }
         }
      }
   }
}
```
