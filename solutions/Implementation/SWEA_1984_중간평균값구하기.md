# SWEA 1984 📊 중간 평균값 구하기 (Java)

## 느낀점
>
- 함수만 알면 쉬운 문제
- 반올림은 `Math.round()`를 사용하면 된다는 것을 다시 확인했다

---

## 흐름
1. 테스트케이스 개수 `T` 입력
2. 각 테스트케이스마다 숫자 10개 입력
3. 배열을 오름차순 정렬
4. 인덱스 `1`부터 `8`까지 더해서 최솟값과 최댓값 제외
5. 합계를 `8`로 나누어 평균 계산
6. `Math.round()`로 반올림 후 출력

---

## 코드

```java
import java.util.Scanner;
import java.io.FileInputStream;
import java.util.Arrays;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);
        int T;
        T=sc.nextInt();

        for(int test_case = 1; test_case <= T; test_case++)
        {
            int[] arr = new int[10];

            for(int i = 0; i < arr.length; i++){
                arr[i] = sc.nextInt();
            }

            Arrays.sort(arr);

            int sum = 0;

            for(int i = 1; i < 9; i++){
                sum += arr[i];
            }

            double average = (double)sum / 8;

            System.out.println("#" + test_case + " " + Math.round(average));
        }
    }
}
```

## 질문 & 실수 정리

### 새로 배운 것

- `Math.round()`는 소수 첫째 자리에서 반올림한 정수 값을 반환한다

---

### 실수했던 부분

- 다만 `Math.round()`의 반환 타입이 `long`이라는 점은 알고 있으면 좋다

---

## AI 코드 리뷰

### 좋았던 점

1. 정렬 활용
   - 최솟값과 최댓값을 쉽게 제외할 수 있어 풀이가 직관적이다

2. 반복 범위가 정확함
   - `for(int i = 1; i < 9; i++)`로 가운데 8개만 더했다

3. 형변환 처리
   - `(double)sum / 8`로 평균을 소수점까지 계산했다

4. 반올림 처리
   - `Math.round(average)`를 사용해서 문제 조건에 맞게 반올림했다

---

### 개선할 점


1. 정렬 없이도 풀이 가능
   - 합계, 최댓값, 최솟값을 입력받으면서 동시에 구하면 배열과 정렬 없이 해결할 수 있다

---

## AI 개선 코드

```java
import java.util.Scanner;

class Solution {
    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();

        for (int test_case = 1; test_case <= T; test_case++) {
            int sum = 0;
            int min = 10000;
            int max = 0;

            for (int i = 0; i < 10; i++) {
                int number = sc.nextInt();

                sum += number;
                min = Math.min(min, number);
                max = Math.max(max, number);
            }

            double average = (double)(sum - min - max) / 8;
            long answer = Math.round(average);

            System.out.println("#" + test_case + " " + answer);
        }
    }
}
```
