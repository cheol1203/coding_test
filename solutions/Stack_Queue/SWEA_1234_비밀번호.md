# SWEA 1234 🔐 비밀번호 (Java)

## 느낀점
>
- 처음에는 `ArrayList`에서 연속된 숫자를 직접 제거하는 방식으로 풀었다
- 하지만 제거할 때마다 리스트가 당겨지고 다시 검사해야 해서 코드가 복잡해졌다
- 스택을 사용하면 현재 숫자와 직전 숫자만 비교하면 되기 때문에 훨씬 간단했다

---

## 흐름
1. 테스트케이스는 총 10개이므로 10번 반복
2. 문자열 길이와 숫자 문자열 입력
3. 숫자를 앞에서부터 하나씩 확인
4. 스택이 비어 있지 않고, 스택 맨 위 숫자와 현재 숫자가 같으면 `pop()`
5. 다르면 현재 숫자를 `push()`
6. 모든 숫자를 확인한 뒤 스택에 남은 숫자를 뒤에서부터 출력
7. 최종 비밀번호 출력

---

## 코드(리스트 사용)

```java
import java.util.Scanner;
import java.util.List;
import java.util.ArrayList;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);

        for(int test_case = 1; test_case <= 10; test_case++)
        {
            int length = sc.nextInt();
            String num = sc.next();

            List<Integer> list = new ArrayList<>();

            for(int i = 0; i < length; i++) {
                list.add(num.charAt(i) - '0');
            }

            boolean isEnd = false;

            while(!isEnd) {
                for(int i = 0; i < list.size() - 1; i++) {
                    if(list.get(i) == list.get(i + 1)) {
                        list.remove(i);
                        list.remove(i);
                        break;
                    }

                    if(i == list.size() - 2) {
                        isEnd = true;
                    }
                }
            }

            System.out.print("#" + test_case + " ");

            for(int i = 0; i < list.size(); i++) {
                System.out.print(list.get(i));
            }

            System.out.println();
        }
    }
}
```

## 제출 코드 (스택사용)

```java
import java.util.Scanner;
import java.util.Deque;
import java.util.ArrayDeque;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);

        for(int test_case = 1; test_case <= 10; test_case++)
        {
            int length = sc.nextInt();
            String num = sc.next();

            Deque<Integer> stack = new ArrayDeque<>();

            for(int i = 0; i < length; i++) {
                int input = num.charAt(i) - '0';

                if(!stack.isEmpty() && stack.peek() == input) {
                    stack.pop();
                } else {
                    stack.push(input);
                }
            }

            System.out.print("#" + test_case + " ");

            while(!stack.isEmpty()) {
                System.out.print(stack.pollLast());
            }

            System.out.println();
        }
    }
}
```
## 질문 & 실수 정리

### 새로 배운 것

- 연속된 같은 숫자를 제거하는 문제는 스택으로 풀 수 있다
- 스택의 맨 위 값은 `peek()`로 확인할 수 있다
- 스택의 맨 위 값을 제거할 때는 `pop()`을 사용한다
- `ArrayDeque`를 `Deque` 타입으로 선언해서 스택처럼 사용할 수 있다
- `push()`로 넣으면 앞쪽에 쌓이기 때문에, 원래 입력 순서대로 출력하려면 `pollLast()`를 사용하면 된다
- `stack.peek() == input`은 `Integer`와 `int` 비교라 자동 언박싱이 일어나서 값 비교가 된다

---

### 실수했던 부분

- 처음에는 `ArrayList`로 직접 삭제하는 방식으로 풀었다
    - 삭제 후 인덱스가 당겨져서 다시 검사해야 하므로 코드가 복잡해졌다
    - 리스트 크기가 0 또는 1이 되면 무한 반복 가능성도 있었다

- 스택 출력 순서를 주의해야 한다
    - `push()`는 앞쪽에 넣는 방식이므로 `pop()`으로 출력하면 역순이 된다
    - 이 문제에서는 `pollLast()`로 뒤쪽부터 꺼내야 원래 순서로 출력된다

- `Integer == Integer` 비교는 참조 비교가 될 수 있지만, 현재 코드는 `Integer == int` 형태라 자동 언박싱으로 값 비교가 된다

## AI 코드 리뷰

### 개선할 점

1. 변수명 개선
    - `num` → `password`
    - `input` → `current`
    - `length`는 그대로 사용해도 좋다

2. `Integer` 대신 `Character` 사용 가능
    - 숫자 연산이 필요한 문제가 아니므로 문자 그대로 비교해도 된다

3. 출력에 `StringBuilder` 사용 가능
    - `System.out.print()`를 여러 번 호출하는 것보다 `StringBuilder`에 모아서 출력하면 더 깔끔하다

---

## AI 개선 코드

```java
import java.util.Scanner;
import java.util.Deque;
import java.util.ArrayDeque;

class Solution {
    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);

        for (int test_case = 1; test_case <= 10; test_case++) {
            int length = sc.nextInt();
            String password = sc.next();

            Deque<Character> stack = new ArrayDeque<>();

            for (int i = 0; i < length; i++) {
                char current = password.charAt(i);

                if (!stack.isEmpty() && stack.peek() == current) {
                    stack.pop();
                } else {
                    stack.push(current);
                }
            }

            StringBuilder answer = new StringBuilder();

            while (!stack.isEmpty()) {
                answer.append(stack.pollLast());
            }

            System.out.println("#" + test_case + " " + answer);
        }
    }
}
```
