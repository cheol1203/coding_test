# SWEA 1244 💰 최대 상금 (Java)

## 느낀점
>
- 첫 dfs 문제, 겁나 어려움
- 처음에는 가장 큰 자리의 숫를 가장 앞으로 옮기는 방법으로 알고리즘을 구상했지만, 부족하다
- 따라서, 모든 경우의 수를 탐색하는 dfs 사용
- 두번째는 모든 교환 경우를 DFS로 탐색하면 될 것 같았지만, 같은 숫자 상태가 반복되면서 시간 초과가 발생했다 -> visited 사용
- 같은 숫자라도 교환 횟수에 따라 의미가 달라지기 때문에 `depth별 HashSet`이 필요하다는 것을 배웠다
- `swap → dfs → swap` 구조가 백트래킹의 핵심이라는 것을 이해했다

---

## 흐름
1. 테스트케이스 개수 `T` 입력
2. 숫자판을 문자열로 입력받고 `char[]`로 변환
3. 교환 횟수 `n` 입력
4. `max`와 `HashSet[]` 초기화
5. DFS로 가능한 모든 자리 교환 탐색
6. 같은 `depth`에서 이미 만든 숫자 상태는 다시 탐색하지 않음
7. 교환 횟수를 모두 사용하면 현재 숫자로 최댓값 갱신
8. 최종 최대 상금 출력

---

## 코드

```java
import java.util.Scanner;
import java.util.HashSet;

class Solution
{
    static int n;
    static char[] numArr;
    static int max;
    static HashSet<String>[] set;

    static void dfs(int depth) {
        if(depth == n) {
            int value = Integer.parseInt(String.valueOf(numArr));
            max = Math.max(value, max);
            return;
        }

        String value = String.valueOf(numArr);

        if(set[depth].contains(value)) {
            return;
        }

        set[depth].add(value);

        for(int i = 0; i < numArr.length - 1; i++) {
            for(int j = i + 1; j < numArr.length; j++) {
                swap(i, j);
                dfs(depth + 1);
                swap(i, j);
            }
        }
    }

    static void swap(int i, int j) {
        char temp = numArr[i];
        numArr[i] = numArr[j];
        numArr[j] = temp;
    }

    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);
        int T;
        T = sc.nextInt();

        for(int test_case = 1; test_case <= T; test_case++)
        {
            numArr = sc.next().toCharArray();
            n = sc.nextInt();

            set = new HashSet[n + 1];

            for(int i = 0; i <= n; i++) {
                set[i] = new HashSet<>();
            }

            max = 0;

            dfs(0);

            System.out.println("#" + test_case + " " + max);
        }
    }
}
```

## 질문 & 실수 정리

### 새로 배운 것

- DFS는 가능한 경우를 깊게 탐색하는 방식이다
- 백트래킹에서는 `swap → dfs → swap`으로 원상복구가 필요하다
- `HashSet`은 중복 여부를 빠르게 확인할 때 사용한다
- `List.contains()`도 가능은 하지만, `HashSet.contains()`가 훨씬 빠르다
- 같은 숫자 상태라도 교환 횟수 `depth`가 다르면 다른 상태로 봐야 한다
- 그래서 `HashSet<String>[]`처럼 depth별 방문 기록을 따로 관리해야 한다

---

### 실수했던 부분

- 처음에는 `max`를 테스트케이스마다 초기화하지 않았다
  - static 변수는 이전 테스트케이스 값이 남을 수 있으므로 매번 `max = 0`이 필요하다
- 단순 DFS로 모든 교환 경우를 탐색해서 시간 초과가 발생했다
  - 같은 숫자 상태가 여러 경로에서 반복해서 만들어지기 때문이다
- 처음에는 `visited`를 왜 depth별로 나눠야 하는지 헷갈렸다
  - `depth = 1`에서의 `321`과 `depth = 2`에서의 `321`은 남은 교환 횟수가 다르므로 다른 상태다

### 개선할 점

1. 변수명 개선
   - `n`은 `changeCount`
   - `numArr`는 `numbers`
   - `set`은 `visited`
   - `max`는 `answer`가 더 의미가 명확하다

2. 방문 체크 위치 조정 가능
   - 현재 코드는 `depth == n`일 때 방문 체크 전에 바로 정답 갱신한다
   - 정답에는 문제 없지만, 흐름상 현재 상태를 먼저 문자열로 만들고 방문 체크 후 종료 조건을 검사하는 방식도 가능하다

3. 제네릭 배열 경고
   - `new HashSet[n + 1]`은 Java에서 unchecked warning이 날 수 있다
   - 코딩테스트에서는 보통 문제 없지만, 실무 코드라면 `ArrayList<HashSet<String>>` 같은 방식도 고려할 수 있다

## AI 개선 코드

```java
import java.util.Scanner;
import java.util.HashSet;

class Solution {
    static int changeCount;
    static char[] numbers;
    static int answer;
    static HashSet<String>[] visited;

    static void dfs(int depth) {
        String current = String.valueOf(numbers);

        if (visited[depth].contains(current)) {
            return;
        }

        visited[depth].add(current);

        if (depth == changeCount) {
            int value = Integer.parseInt(current);
            answer = Math.max(answer, value);
            return;
        }

        for (int i = 0; i < numbers.length - 1; i++) {
            for (int j = i + 1; j < numbers.length; j++) {
                swap(i, j);
                dfs(depth + 1);
                swap(i, j);
            }
        }
    }

    static void swap(int i, int j) {
        char temp = numbers[i];
        numbers[i] = numbers[j];
        numbers[j] = temp;
    }

    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();

        for (int test_case = 1; test_case <= T; test_case++) {
            numbers = sc.next().toCharArray();
            changeCount = sc.nextInt();

            answer = 0;
            visited = new HashSet[changeCount + 1];

            for (int i = 0; i <= changeCount; i++) {
                visited[i] = new HashSet<>();
            }

            dfs(0);

            System.out.println("#" + test_case + " " + answer);
        }
    }
}
```
