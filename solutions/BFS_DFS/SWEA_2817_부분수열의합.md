# SWEA 2817 🔢 부분 수열의 합 (Java)

## 느낀점
>
- 5215번 햄버거 다이어트와 비슷하게 각 원소를 선택하거나 선택하지 않는 DFS 문제였다
- 목표 합 `K`가 정해져 있어서, 현재 합이 `K`보다 커지면 더 탐색할 필요가 없었다
- 합이 정확히 `K`가 되는 순간 `count++` 하고 탐색을 종료하면 된다

---

## 흐름
1. 테스트케이스 개수 `T` 입력
2. 숫자 개수 `N`, 목표 합 `K` 입력
3. 숫자 배열 입력
4. DFS로 각 숫자를 선택하는 경우와 선택하지 않는 경우를 탐색
5. 현재 합이 `K`보다 크면 탐색 중단
6. 현재 합이 `K`와 같으면 경우의 수 증가
7. 모든 탐색 후 `count` 출력

---

## 코드

```java
import java.util.Scanner;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);
        int T;
        T=sc.nextInt();

        for(int test_case = 1; test_case <= T; test_case++)
        {
            int n = sc.nextInt();
            k = sc.nextInt();

            arr = new int[n];

            for(int i = 0 ; i < n; i++) {
                arr[i] = sc.nextInt();
            }

            count = 0;
            dfs(0, 0);

            System.out.println("#" + test_case + " " + count);
        }
    }

    static int k;
    static int count;
    static int[] arr;

    static void dfs(int index, int sum) {
        if(sum > k) {
            return;
        } else if(sum == k) {
            count++;
            return;
        }

        if(index == arr.length) {
            return;
        }

        dfs(index + 1, sum + arr[index]);
        dfs(index + 1, sum);
    }
}
```

## 질문 & 실수 정리

### 실수했던 부분

- 이번 코드는 큰 실수 없이 잘 작성했다
- 다만 `sum == k`일 때 바로 `return`하는 방식은 이 문제에서는 괜찮다
    - SWEA 2817은 자연수 배열이기 때문에 뒤에 숫자를 더하면 합이 더 커질 뿐이다
    - 따라서 이미 `K`가 된 순간 더 탐색하지 않아도 된다

- 만약 배열에 `0`이나 음수가 포함되는 문제라면 `sum == k`에서 바로 종료하면 안 될 수도 있다
    - 뒤에 `0`을 추가해도 합이 유지될 수 있고, 음수로 다시 줄어들 수도 있기 때문이다
    - 하지만 SWEA 2817 조건에서는 양의 정수라서 현재 방식이 가능하다

## AI 코드 리뷰

### 개선할 점

1. 변수명 개선
    - `k` → `target`
    - `arr` → `numbers`
    - `count` → `answer`

2. `n`도 전역 변수로 관리 가능
    - 지금처럼 `arr.length`를 써도 문제 없지만, `n`을 전역 변수로 두면 의미가 더 명확할 수 있다

3. 종료 조건 순서 정리 가능
    - 보통은 `index == arr.length`에서 합이 목표값인지 확인하는 형태도 많이 쓴다
    - 하지만 현재 코드는 자연수 조건을 활용해 더 빠르게 종료하는 방식이라 괜찮다

---

## AI 개선 코드

```java
import java.util.Scanner;

class Solution {
    static int target;
    static int answer;
    static int[] numbers;

    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();

        for (int test_case = 1; test_case <= T; test_case++) {
            int n = sc.nextInt();
            target = sc.nextInt();

            numbers = new int[n];

            for (int i = 0; i < n; i++) {
                numbers[i] = sc.nextInt();
            }

            answer = 0;
            dfs(0, 0);

            System.out.println("#" + test_case + " " + answer);
        }
    }

    static void dfs(int index, int sum) {
        if (sum > target) {
            return;
        }

        if (sum == target) {
            answer++;
            return;
        }

        if (index == numbers.length) {
            return;
        }

        dfs(index + 1, sum + numbers[index]);
        dfs(index + 1, sum);
    }
}
```
