# SWEA 2001 🪰 파리 퇴치 (Java)

## 느낀점
>
- 한번에 맞춘 문제
- 기본 완전탐색 문제였지만 2중 반복 안에 또 2중 반복이 들어가서 처음엔 구조 잡는 게 조금 헷갈렸다
- 불필요한 배열을 만들었다가 다시 지우면서 문제를 단순하게 보는 게 중요하다는 걸 느낌
- 구현 자체는 어렵지 않지만, 반복문의 범위를 정확하게 잡는 게 중요했다

---

## 흐름
1. 바깥 2중 for문은 `M x M` 영역의 이동
2. 안쪽 2중 for문은 `M x M` 영역의 탐색 후 해당 영역의 합을 구함
3. 바깥 2중 for문 한번 돌때마다 최댓값 갱신

---

## 코드
```java
import java.util.Scanner;

class Solution
{
    public static void main(String args[]) throws Exception
    {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();

        for(int test_case = 1; test_case <= T; test_case++)
        {
            int n = sc.nextInt();
            int m = sc.nextInt();
            
            int[][] n_arr = new int[n][n];
            int max = 0;

            for(int i = 0; i < n; i++){
                for(int j = 0; j < n; j++){
                    n_arr[i][j] = sc.nextInt();
                }
            }
                    
            for(int i = 0; i < n - m + 1; i++){
                for(int j = 0; j < n - m + 1; j++){
                    int sum = 0;

                    for(int k = i; k < i + m; k++){
                        for(int w = j; w < j + m; w++){
                            sum += n_arr[k][w];
                        }
                    }

                    if(sum > max){
                        max = sum;
                    }
                }
            }

            System.out.println("#" + test_case + " " + max);
        }
    }
}
```

## 개선된 코드

```java
import java.util.Scanner;

class Solution {
    public static void main(String args[]) throws Exception {
        Scanner sc = new Scanner(System.in);

        int T = sc.nextInt();

        for (int testCase = 1; testCase <= T; testCase++) {
            int N = sc.nextInt();
            int M = sc.nextInt();

            int[][] flies = new int[N][N];

            for (int i = 0; i < N; i++) {
                for (int j = 0; j < N; j++) {
                    flies[i][j] = sc.nextInt();
                }
            }

            int maxFlies = 0;

            for (int i = 0; i <= N - M; i++) {
                for (int j = 0; j <= N - M; j++) {
                    int currentSum = 0;

                    for (int r = i; r < i + M; r++) {
                        for (int c = j; c < j + M; c++) {
                            currentSum += flies[r][c];
                        }
                    }

                    maxFlies = Math.max(maxFlies, currentSum);
                }
            }

            System.out.println("#" + testCase + " " + maxFlies);
        }

        sc.close();
    }
}
```
